# CareerTrack 🚀

CareerTrack is a full-stack placement and internship management application that helps students organize and monitor their placement journey. It allows users to track applications, manage interview progress, monitor deadlines, and visualize placement statistics through an interactive dashboard.

---

## ✨ Features

- Add, edit, delete, and manage placement/internship applications
- Search and filter applications
- Track application status (Applied, OA, Technical, HR, Offer, Rejected)
- Activity History for every application
- Interview Journal to record interview experiences
- Dynamic Days Remaining calculation for deadlines
- Dashboard with analytics and charts
- Statistics page for placement insights
- Responsive dark-themed UI

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Sharvani-Shriya/CareerTrack.git
```

### Navigate into the project

```bash
cd CareerTrack
```

### Install dependencies

```bash
npm install
```

### Start MongoDB Community Server

Make sure MongoDB is running locally on:

```
mongodb://127.0.0.1:27017/careertrack
```

### Start the application

```bash
npm start
```

Visit:

```
http://localhost:3000
```
